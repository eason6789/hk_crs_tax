# 富途 CRS 境外所得税计算器

[![Python](https://img.shields.io/badge/Python-3.8+-blue.svg)](https://www.python.org/)

> 🧾 完全本地运行 | 不收费 | 无页数限制 | 隐私安全

解析富途证券( Futu/Moomoo )年度账单 Excel，按中国个人所得税法自动计算境外股票/基金/期权的转让所得、股息红利、利息收入，输出分市场(港股/美股/基金)的应补税额。

## ✨ 功能特色

- 🆓 **完全免费** — 本地 Python 脚本运行，无需注册、无需 API Key、无页数限制
- 🔒 **隐私安全** — 数据仅在本地处理，不上传任何服务器
- 📊 **分市场统计** — 港股🇭🇰、美股🇺🇸、基金🌐 分别计算盈亏
- 🧮 **双成本法** — 支持移动加权平均法(WMA)和先进先出法(FIFO)
- 📑 **专业输出** — 生成含 PQR 列的 Excel 审计底稿，盈利绿色/亏损红色
- 📅 **跨年结转** — 自动从上年度交易记录结转持仓成本
- 💱 **汇率折算** — 自动按纳税年度末日人民币中间价折算
- 💵 **股息抵免** — 自动识别境外预扣税并计算税收抵免
- ⚠️ **风险提示** — 自动标记 RSU/IPO 等需要手动确认成本的项目

## 📋 快速开始

### 前提条件

```bash
pip install openpyxl
```

### 基本用法

```bash
# 单年度计算（推荐同时提供上年文件以获得准确成本）
python3 scripts/futu_tax_calculator.py \
  2025_年度账单.xlsx \
  2024_年度账单.xlsx \
  -o output/2025_税务审计底稿.xlsx

# 批量处理所有年度
python3 scripts/futu_tax_calculator.py --all-years /path/to/bills/

# 使用 FIFO 方法
python3 scripts/futu_tax_calculator.py 2025_年度账单.xlsx --method FIFO

# 手动指定汇率
python3 scripts/futu_tax_calculator.py 2025_年度账单.xlsx \
  --usd-rate 7.1884 --hkd-rate 0.9246
```

## 🧮 计算原理

### 移动加权平均法 (WMA, 默认)

符合财税〔2016〕36号文，是当前中国税务机关认可的方法：

- **P 列（持仓股数）**：买入累加，卖出扣减，滚动求和
- **Q 列（加权成本）**：每次买入后重新计算 `Q = (旧总成本 + 新买入金额) ÷ 新总股数`，卖出时 Q 不变
- **R 列（卖出盈亏）**：`R = (卖出价 − Q) × 股数 − 卖出手续费`

### 先进先出法 (FIFO)

按买入批次顺序匹配卖出，`--method FIFO` 启用。

### 税务公式

```
资本利得税 = max(0, 年度净盈亏 × 汇率 × 20%)
股息红利税 = max(0, 股息总额 × 20% − 境外已预扣税额)
利息收入税 = 利息/票息收入 × 20%
```

## ⚠️ 重要提醒

1. **申报时间**：次年 3月1日至6月30日，逾期按日加收万分之五滞纳金
2. **浮盈不征税**：仅对实际卖出实现的盈利计税
3. **RSU 成本**：RSU 归属时已按工资薪金缴税，卖出时以归属日市价作为成本基准
4. **保留凭证**：建议保留完整交易记录和完税证明至少 5 年
5. **汇率**：实际申报时请以人民银行公布的纳税年度末日中间价为准
6. **仅供参考**：本工具计算结果仅供辅助参考，不构成税务建议

## 🔧 关于 CRS Tax Calculator API

市面上有些工具（如 wealthlplantation.com）提供云端 CRS 计算服务，通常有页数限制（免费 20 页，超出需付费）。本工具完全在本地运行，**无需联网、不收费、无页数限制**，适合注重隐私和不愿额外付费的用户。
