# 境外券商 CRS 个人所得税计算器 v4.0

## 支持的文件格式

| 格式 | 来源 | 处理方式 |
|------|------|---------|
| `.xlsx` 年度账单 | 富途App→我的→税表→CRS | `scripts/calculate_tax.py` 直接解析 |
| `.xlsx` 月结单 | 券商月度对账单 | 自动检测sheet名和列名，兼容多种格式 |
| `.pdf` 月结单 | 券商PDF对账单 | `pdfplumber`/`PyPDF2` 本地解析提取文本 |
| `.csv` 交易记录 | 券商导出CSV | `csv.DictReader` 自动检测列名 |
| `.png/.jpg` 截图 | 对账单截图 | OCR提取（需`tesseract`，或引导用户用Excel） |

## 支持的券商

富途(Futu/Moomoo)、长桥(Longbridge)、老虎(Tiger)、盈透(IBKR)、盈立(uSMART)、卓锐(Zircon) — 自动从文件内容识别。

## 支持的品种

股票、基金(申购/赎回)、期权(卖出开仓/买入平仓)、期货、债券、结构化票据(FCN)

## 工作流程

### 第一步：确认输入

询问用户：
- 要处理的文件路径（支持多个文件混合格式）
- 纳税年度（默认从文件名提取）
- 是否需要提供上年度文件（跨年持仓结转成本）

### 第二步：运行计算

```bash
# 单年度（推荐同时提供上年）
python3 scripts/calculate_tax.py 2025_年度账单.xlsx 2024_年度账单.xlsx

# 多年月结单合并为一年
python3 scripts/calculate_tax.py 1月.xlsx 2月.pdf 3月.csv --year 2025

# 批量处理
python3 scripts/calculate_tax.py --all-years /path/to/dir/

# 可选参数
--method FIFO     # 先进先出法（默认WMA移动加权平均）
--usd-rate 7.2    # 手动指定汇率
--hkd-rate 0.92
```

### 第三步：解读结果

输出 Excel 包含5个sheet：交易明细(含PQR列)、持仓汇总、税务报告(分市场)、股息利息明细、需确认事项。

## 计算逻辑

### P/Q/R 列定义

- **P（持仓股数）**：买入累加、卖出扣减的滚动求和。归零后重新开始。
- **Q（加权成本）**：移动加权平均法。每次买入后：`新总成本÷新总股数`。卖出时不变。买入手续费计入成本（依据《个人所得税法》财产原值定义）。
- **R（卖出盈亏）**：`(卖出价 − Q) × 卖出股数 − 卖出手续费`。仅卖出时计算。

### 交易方向识别

| 方向 | 品种 | 操作 |
|------|------|------|
| 买入开仓 | 股票/基金 | 买入，更新P/Q |
| 卖出平仓 | 股票/基金 | 卖出，计算R |
| 申购 | 基金 | 同买入 |
| 赎回 | 基金 | 同卖出 |
| 卖出开仓 | 期权 | 开空仓，收权利金 |
| 买入平仓 | 期权 | 平空仓，盈亏=开仓费−平仓费 |
| 买入开仓 | 期权 | 开多仓 |
| 卖出平仓 | 期权 | 平多仓 |

### RSU/IPO 成本处理

非交易获得的资产（RSU归属、IPO中签）：
1. 优先查 `rsu_config.json` 中的 FMV 配置
2. 从税款记录反推：`FMV = 扣税金额 ÷ 税率 ÷ 股数`
3. 从附近交易价格估算
4. 以上都失败则标记为需手动确认

### 税务公式

```
资本利得税 = max(0, 年度净盈亏 × 汇率 × 20%)  # 同市场同币种可互抵
股息红利税 = max(0, 股息总额×20% − 境外已预扣税)  # 预扣税可抵免
利息收入税 = 利息/票息收入 × 20%
```

2024+2025年盈利才需补税，2021-2023亏损年份无需补。
申报时间：次年3/1-6/30，逾期日万分之五滞纳金。

### RSU 双重征税问题

RSU归属时已按工资薪金缴税（雇主代扣），卖出时以归属日市价(FMV)为成本计算资本利得。两者是两个独立税种，不存在重复征税。

## 汇率

优先从 `exchangerate-api.com` 自动获取纳税年度末日中间价，失败则提示用户手动输入。

## 成本方法选择

- **WMA（移动加权平均，默认）**：财税〔2016〕36号明确支持，适合频繁交易
- **FIFO（先进先出）**：部分实操使用，早期高成本先卖出→亏损更大→税更少

## 参考法规

- 《个人所得税法》(2018修正)
- 财税〔2016〕36号 — 金融商品转让成本核算
- 财税2020年第3号 — 境外所得申报
- 财税〔2009〕5号/国税函〔2009〕461号 — RSU计税
